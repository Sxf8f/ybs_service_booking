Excel File Format Requirements

Required Format

Your Excel file must have the following columns in this exact order:

# Column Name Description Example
1 Order ID Unique order identifier 2567377108
2 Order Date Date of order (DD.MM.YYYY or YYYY-MM-DD) 19.10.2025
3 Partner ID Partner/Retailer identification number 0660143354
4 Partner Name Retailer name (must match exactly) ALMAS MOBILES
5 Transfer Amount Total transfer amount 2060
6 Commission Commission amount 60
7 Amount Without Commission Amount retailer owes (Transfer - Commission) 2000
Important Rules:
  • Column names must match exactly (case-insensitive)
  • Partner Name must match a registered retailer under the selected FOS
  • Order ID must be unique (duplicates will be skipped)
  • All amounts must be numeric (no currency symbols)
  • File must be .xlsx or .xls format
